第一节 网络基本概念
============================

发展阶段
------------------

1. ARPANET（1969， 美国， 第一个） -> 互联网

2. 三级结构的互联网 （主干网，地区网，校园网）

3. 多层次ISP结构的互联网

.. note::

   ISP 互联网服务提供商，任何机构或个人都可以向ISP缴纳费用取得IP地址的使用权

概念以及组成
---------------------


概念
~~~~~~~~~~~~~

由一些通用的，可编程的硬件互连而成

组成
~~~~~~~~~~~

1. 物理组成

   硬件、软件、协议

   硬件：主机、通信线路、交换设备

   软件：实现资源共享的软件

   协议：一种为进行网络中的数据交换而建立的规则

2. 功能组成

   通信子网、资源子网

   通信子网：有各种传输介质、通信设备和相关网络协议组成
